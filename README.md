# 👑 Problema das N-Rainhas com Algoritmo Genético

Este repositório contém uma implementação em **Java** para resolver o clássico desafio das **N-Rainhas** utilizando **Algoritmos Genéticos (AG)**. O projeto foi desenvolvido como parte prática da disciplina de **Inteligência Artificial I**.

## 📝 O Problema
O objetivo é posicionar $N$ rainhas em um tabuleiro de xadrez de tamanho $N \times N$ de forma que nenhuma rainha ataque outra. Isso significa que não pode haver duas rainhas na mesma linha, coluna ou diagonal.
